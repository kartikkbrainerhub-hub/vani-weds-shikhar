# Wedding Invitation Web App

A premium, mobile-first wedding invitation web application featuring an immersive wax-seal envelope opening animation and scroll-based storytelling.

## ✨ Features

- **Immersive Opening Animation**: A full-screen, interactive wax-seal envelope that opens upon tapping.
- **Mobile-First Design**: Optimized for smartphones with a focus on immersive, distraction-free UI.
- **Scroll-Based Storytelling**: Narrates the couple's story, event details, and location through smooth scrolling.
- **Gallery & Countdown**: Beautifully displayed memories and a real-time countdown to the big day.
- **RSVP Integration**: Simplified RSVP form for guests.
- **Floating Petals**: Subtle, elegant background animations for a romantic atmosphere.
- **Minimalist Music Player**: Floating, non-intrusive music controls.

## 🛠️ Tech Stack

- **Frontend**: React, Framer Motion, Tailwind CSS, Lucide React.
- **Backend**: FastAPI (Python), SQLite.
- **Styling**: Vanilla CSS + Tailwind utilities.

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- Python (v3.9+)

### Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd invt-card
   ```

2. **Frontend Setup**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**:
   ```bash
   cd ../backend
   pip install -r requirements.txt
   python main.py
   ```

## 📂 Project Structure

- `/frontend`: React application containing components, assets, and styling.
- `/backend`: FastAPI server handling RSVP data and database interactions.

## 🎨 Configuration

Update `frontend/src/config.js` to customize:
- Names of the couple
- Wedding date and time
- Venue details and map links
- Story and gallery images
